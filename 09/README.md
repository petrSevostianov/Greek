## Упражнения с глаголами группы Β
### 1. Вставь правильную форму глагола в скобках
(Β1 и Β2)
- Εσύ **μιλάς** (μιλάω) ελληνικά. `β1` = Говоришь по-гречески.
- Εσύ **αργείς** (αργώ) κάθε μέρα στη δουλειά. `β2` = Опаздываешь каждый день на работу.
- Αυτός δεν **αγαπάει** (αγαπάω) τα ψέματα. `β1` 
- Εμείς **τηλεφωνούμε** (τηλεφωνώ) στους φίλους μας συχνά. `β2`
- Εσείς **χαμογελάτε** (χαμογελάω) πολύ όμορφα! `β1` = Улыбаетесь очень красиво!
- Αυτοί **απαντούν** (απαντώ) γρήγορα στα e-mail. `β2` = Отвечают быстро на e-mail.

```
αργώ = опаздывать
αγαπάω = любить
χαμογελάω = улыбаться
απαντάω = отвечать
```

### 2. Соедини стрелками глагол с правильной формой местоимения

### 3. Исправь ошибки в предложениях
 - Αυτή ~~μιλάω~~ μιλάει ελληνικά πολύ καλά.
 - Εσείς ~~απανταώ~~ απαντάτε στις ερωτήσεις. = Вы отвечаете на вопросы.
 - Εγώ ~~αγαπάς~~ αγαπάω τις γάτες. = Я люблю кошек.
 - Αυτοί ~~μπορεί~~ μπορούν να ~~τρώει~~ τρώνε πολύ. = Они могут много есть.
 - Εμείς ~~αργείς~~ αργούμε κάθε πρωί. = Мы опаздываем каждое утро.

```
вопрос = ερώτηση
```

### 4. Заполни пропуски нужной формой глагола
 - Οι φίλοι μου μιλάνε (μιλάω) αγγλικά και ελληνικά.
 - Η Μαρία μπορεί (μπορώ) να τραγουδήσει καλά. = Мария может хорошо петь.
 - Εγώ χαμογελάω (χαμογελάω) όταν σε βλέπω! = Я улыбаюсь, когда тебя вижу!
 - Εσύ τηλεφωνείς (τηλεφωνώ) κάθε μέρα στη μητέρα σου; = Ты звонишь каждый день маме твоей?
 - Τα παιδιά αγαπάνε (αγαπάω) το καλοκαίρι.

```
όταν = когда
```

### 5. Выбери правильную форму глагола
 - Αυτός (~~απαντώ~~ / απαντάει) σωστά στις ερωτήσεις. = Он правильно отвечает на вопросы.
 - Εμείς (~~τηλεφωνείτε~~ / τηλεφωνούμε) στους γονείς μας. = Мы звоним нашим родителям.
 - Εσύ (αγαπάς / ~~αγαπαώ~~) το παγωτό σοκολάτα; = Ты любишь мороженое шоколадное?
 - Αυτοί (~~μπορούμε~~ / μπορούν) να έρθουν αύριο.
 - Εγώ (μιλαώ / ~~μιλάς~~) πολύ καλά γερμανικά.

### 6. Поставь глаголы в правильной форме
Εσύ δεν μιλάς (μιλαώ) ισπανικά.
Εγώ δεν μπορώ (μπορώ) να έρθω αύριο. = Я не могу прийти завтра.
Η Άννα δεν απαντάει (απανταώ) στα μηνύματα. = Анна не отвечает на сообщения.
Αυτοί δεν αγαπάνε (αγαπάω) το χειμώνα. = Они не любят зиму.
Εσείς δεν χαμογελάτε (χαμογελάω) συχνά. = Вы не улыбаетесь часто.

```
απαντάει = отвечает
χειμώνα = зима
```

### 7. Напиши вопросы, используя данные слова
 - (εσύ / αγαπάω / μουσική) → Εσύ αγαπάς τη (την) μουσική;
 - (εσείς / μιλάω / ελληνικά) → Εσείς μιλάτε ελληνικά;
 - (αυτός / απανταώ / στην ερώτηση) → Αυτός απαντάει στην ερώτηση;
 - (εμείς / χαμογελάω / συχνά) → Eμείς χαμογελάμε συχνά;

```
η μουσική = музыка
οι μουσικές = музыки
```

## Вставьте правильные местоимения  

Родительный падеж точно такой же, как и притяжательные.

**Родительный = притяжательные**
| _ | Притяжательные (Чей?) | Личные винительный (Кого?) | Личные родительный (Кому?) |
|----|----|----|----| 
| Εγώ | Мой **Μου** | Меня **Με** | Мне **Μου** |
| Εσύ | Твой **Σου** | Тебя **Σε** | Тебе **Σου** |
| Αυτός | Его **Του** | Его **Τον** | Ему **Του** |
| Αυτή | Её **Της** | Её **Την** | Ей **Της** |
| Αυτό | Его (ср) **Του** | Его, это, it (ср) **Το** | Ему, этому (ср.) **Του** |
| Εμείς | Наш **Μας** | Нас **Μας** | Нам **Μας** |
| Εσείς | Ваш **Σας** | Вас **Σας** | Вам **Σας** |
| Αυτοί | Их (м) **Τους** | Их (м) **Τους** | Им (м) **Τους** |
| Αυτές | Их (ж) **Τους** | Их (ж) **Τις** | Им (ж) **Τους** |
| Αυτα | Их (ср) **Τους** | Их (ср) **Τα** | Им (ср) **Τους** |



**Του** δίνω το βιβλίο τώρα. = Я отдаю ему книгу сейчас.

Τα παιδιά **τον** ψάχνουν στον κήπο. = Дети ищут его (мужчину) в саду.

Ο δάσκαλός μου **μου** βοηθάει με τα μαθήματα. = Мой учитель помогает мне с уроками.

```
βοηθάω = помогать
βοηθάει = помогает
```

Η γιαγιά **τού** στέλνει το γράμμα. = Бабушка отправляет письмо ему.

**Την** θυμάσαι από το καλοκαίρι; = Ты помнишь её с лета?

Ο Γιάννης μας **της** δίνει το δώρο τώρα. = Яннис наш отдаёт подарок ей сейчас.

**Την** περιμένω στο σταθμό. = Я жду её на станции.

**Τους** άκουσα στην τηλεόραση χθες. = Я слышал их вчера по телевизору.

```
χθες = вчера
```


Η μαμά μας **μας** μαγειρεύει φαγητό. = Наша мама готовит еду для нас.

Μου αρέσει να **την** διαβάζω κάθε βράδυ.  (εφημερίδα - газета) = Мне нравится читать её каждый вечер.

Θέλω να **της** δείξω τις φωτογραφίες μου. = Я хочу показать ей свои фотографии.

```
δείξω = показать
```

Ο δάσκαλος μας **μας** εξηγεί τα πάντα. = Наш учитель объясняет всё нам.

```
εξηγώ = объяснять
```

Η μαμά σας **μας** αγαπάει πολύ. = Ваша мама очень любит нас.

Το παιδί μας **τους** μιλάει για το σχολείο. = Наш ребёнок рассказывает им о школе.

**Τον** βλέπω κάθε μέρα στο σχολείο. = Я вижу его каждый день в школе.

Πρέπει να **τους** ευχαριστήσουμε για το δώρο. (друзей) = Нам нужно поблагодарить их за подарок.

```
Πρέπει = нужно, следует
```

Ο φίλος μου πάντα **μου** λέει την αλήθεια. = Мой друг всегда говорит мне правду.
```
λέω = говорю
λέει = говорит
αλήθεια = правда
```


**Της** δίνουμε λουλούδια στα γενέθλια της. = Мы дарим ей цветы на её день рождения.

```
γενέθλια = день рождения
```

**Της** μιλάμε στο τηλέφωνο κάθε μέρα. = Мы говорим ей по телефону каждый день.

Ο Γιώργος θέλει να **μας** συναντήσει αύριο. = Георгиос хочет встретить нас завтра.

Εργάζομαι (Работать)

- Αυτός **εργάζεται** σε ένα γραφείο. - Он работает в офисе.
- Εμείς **εργαζόμαστε** σκληρά. - Мы работаем усердно.
- Εσυ **εργάζεσαι** από το σπίτι. - Ты работаешь из дома.
- Αυτοί **εργάζονται** σε μια εταιρεία. - Они работают в компании.


Ξεκουράζομαι (Отдыхать) Г1

- Αυτή **Ξεκουράζεται** μετά τη δουλειά. - Она отдыхает после работы.
- Εσείς **Ξεκουράζεστε** στον κήπο. - Вы отдыхаете в саду.

- Αυτός **φαντάζεται** ένα όμορφο μέλλον. - Он представляет себе красивое будущее.
- Εσείς **φαντάζεστε** πολλά πράγματα. - Вы представляете себе много вещей.
