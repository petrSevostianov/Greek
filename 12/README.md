## Изменение окончаний существительных в родительном падеже
|  Род  | Именительный падеж | Родительный падеж | Пример |
|-----|-----------------|------------------|-----------------|
| М | -ος | -ου | ο φίλος – του φίλου |
|   | -ας | -α | ο μπαμπάς – του μπαμπά |
|   | -ης | -η | ο μαθητής – του μαθητή |
| Ж | -α | -ας | η Μαρία – της Μαρίας |
|   | -η | -ης | η Ελένη – της Ελένης |
|   | -ος | -ου | η τσάντα – της τσάντας |
| Ср | -ο | -ου | το βιβλίο – του βιβλίου |
|   | -ι | -ιου | το παιδί – του παιδιού |
|   | -μα | -ματος | το όνομα – του ονόματος |
|  | -ος | -ους | τα παράθυρα – των παραθύρων |

## Στο σπίτι της Μαρίας = В доме Марии
Η Μαρία ζει σε ένα όμορφο σπίτι στο κέντρο **της πόλης** (η πόλη).

Το σπίτι **της Μαρίας**  (η Μαρία) είναι δίπλα στο πάρκο.

Η είσοδος **του σπιτιού** (το σπίτι) είναι μεγάλη και η πόρτα **της εισόδου** (η είσοδος) είναι μπλε. = Вход дома большой, а дверь входа синяя.

Η Μαρία αγαπάει τα λουλούδια **του κήπου** (ο κήπος) **της Μαρίας** (η Μαρία).

Κάθε πρωί ποτίζει τα λουλούδια με το νερό **του πηγαδιού** (το πηγάδι).

Στο σαλόνι **του σπιτιού** (το σπίτι), υπάρχει μια μεγάλη βιβλιοθήκη. = В гостиной дома есть большая библиотека.

Τα βιβλία **της Μαρίας** (η Μαρία) είναι πολλά και διαβάζει κάθε βράδυ. =  Книг Марии много, и она читает каждый вечер.

Το αγαπημένο το βιβλίο είναι μια ιστορία **της Ελλάδας** (η Ελλάδα).

Στον τοίχο **της κουζίνας** (η κουζίνα), κρέμεται ένας πίνακας **της οικογένειας** (η οικογένεια) **Της Μαρίας** (η Μαρία). = На стене кухни висит картина семьи Марии.

Η Μαρία μαγειρεύει κάθε μέρα και το αγαπημένο το φαγητό είναι η πίτα **της γιαγιάς** (η γιαγιά) **της Μαρίας** (η Μαρία). = Мария готовит каждый день, и любимое блюдо - пирог бабушки Марии.

Το βράδυ, η Μαρία πηγαίνει στο δωμάτιο **του ύπνου** (ο ύπνος). = Вечером Мария идет в комнату сна.

Η κρεβατοκάμαρα **της Μαρίας** (η Μαρία) είναι άνετη και έχει ένα παράθυρο με θέα τη θάλασσα. = Спальня Марии удобная и имеет окно с видом на море.

Το φως **του φεγγαριού** (το φεγγάρι) γεμίζει το δωμάτιο και η Μαρία κοιμάται γλυκά. = Свет луны наполняет комнату, и Мария спит сладко.
